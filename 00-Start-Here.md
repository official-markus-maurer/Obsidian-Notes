## 🛠️ Prerequisites

Before you start, make sure you have a C compiler installed.
- **Windows**: MinGW or Visual Studio Community.
- **Mac**: Xcode Command Line Tools (`xcode-select --install`).
- **Linux**: GCC (`sudo apt install build-essential`).

## 📚 How to use this Vault

- **Read**: Go through the notes in order.
- **Code**: Don't just read! Type out the examples in your code editor and run them.
- **Modify**: Experiment with the code. What happens if you change a value?
- **Link**: Create your own connections between concepts.
