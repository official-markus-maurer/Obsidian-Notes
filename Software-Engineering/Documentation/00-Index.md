# Documentation with Doxygen

Writing code is half the battle; explaining it is the other half. In C, **Doxygen** is the standard tool for generating documentation from source code comments.

## Topics
1.  [[01-Doxygen-Basics]]: Setting up Doxygen and `Doxyfile`.

## Why Document?
-   **API Reference**: Auto-generate HTML/PDF manuals for your library users.
-   **Maintenance**: Helps new developers understand the codebase.
