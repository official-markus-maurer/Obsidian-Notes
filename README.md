# The C Programming Vault 🏛️

A comprehensive, zero-to-hero curriculum for mastering the C programming language and its ecosystem.

## 📖 Overview
This repository is structured as an **Obsidian Vault**, but it works perfectly as a standard GitHub repository. It covers not just the C language, but the entire engineering ecosystem around it.

### 📂 Directory Structure

-   **`C-Programming/`**: The core curriculum.
    -   Fundamentals, Pointers, Systems Programming, Advanced Topics.
-   **`DevOps/`**: Tools for deploying and managing C applications.
    -   Docker, CI/CD, Binary Artifacts.
-   **`Software-Engineering/`**: General engineering practices applied to C.
    -   Build Systems, Documentation, Version Control.
-   **`CS-Fundamentals/`**: The theory behind the code.
    -   Data Structures, Algorithms, Computer Architecture.
-   **`Mathematics/`**: Applied Math for CS.
    -   Linear Algebra (Graphics), Discrete Math (Logic).
-   **`Shared-Concepts/`**: Cross-cutting knowledge.
    -   Memory Management, Concurrency.
-   **`Code-Examples/`**: Compilable, practical code examples.

## 🚀 Getting Started

### Prerequisites
-   A C Compiler (`gcc`, `clang`, or `cl`).
-   `make` (optional, for building examples).

### Building Examples
A collection of compilable examples is located in `Code-Examples/`.

```bash
# Build all examples
make

# Build specific example
make basics
make fsm
make libmath
```
