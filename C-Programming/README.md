# The C Programming Vault 🏛️

A comprehensive, zero-to-hero curriculum for mastering the C programming language.

## 📖 Overview
This repository is structured as an **Obsidian Vault**. It is designed to be read in Obsidian, but the markdown files are perfectly readable on GitHub.

The curriculum is divided into **6 Phases**:
1.  **Fundamentals**: Syntax, variables, control flow.
2.  **Intermediate**: Pointers, memory management, data structures.
3.  **Systems Programming**: Unix/Linux system calls, signals, processes.
4.  **Advanced Topics**: Interop, SIMD, functional patterns.
5.  **Professional Development**: GUI, Games, Kernel dev.
6.  **Engineering Standards**: Build systems, CI/CD, documentation.

## 🚀 Getting Started

### Prerequisites
-   A C Compiler (`gcc`, `clang`, or `cl`).
-   `make` (optional, for building examples).

### Building Examples
A collection of compilable examples is located in `Code-Examples/`.

```bash
# Build all examples
make

# Build specific example
make basics
make fsm
make gui
```

## 📂 Project Structure
-   `00-Start-Here.md`: The main entry point and roadmap.
-   `XX-ModuleName/`: Each module contains markdown notes.
-   `Code-Examples/`: Practical C code demonstrating the concepts.
