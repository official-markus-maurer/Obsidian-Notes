# Project Ideas

Building projects is the best way to learn. Here are ideas graded by difficulty.

## 🟢 Beginner
1.  **Linked List Library**: Implement generic double linked list.
2.  **Simple Shell**: Read input, parse command, `fork` and `exec`.
3.  **Text Editor**: Open a file, edit buffer, save (like `nano`).

## 🟡 Intermediate
1.  **HTTP Server**:
    -   Parse GET requests.
    -   Serve static files.
    -   Implement concurrent handling (Threads or `select`).
2.  **Chat Application**:
    -   Client-Server architecture.
    -   Broadcasting messages to all clients.
3.  **JSON Parser**: Write your own JSON tokenizer and parser.

## 🔴 Advanced
1.  **Memory Allocator**:
    -   Implement `malloc`, `free`, `realloc`.
    -   Use `sbrk` or `mmap`.
    -   Handle fragmentation.
2.  **Key-Value Store**:
    -   Persistent storage (B-Tree or LSM Tree).
    -   Network interface (Redis clone).
3.  **Game Boy Emulator**:
    -   Emulate the Z80-like CPU.
    -   Render graphics via SDL/OpenGL.
