This section covers more complex features of C that give you fine-grained control over the program and system.

4. **[[04-Command-Line-Arguments|Command Line Arguments]]**

5. **[[05-Makefiles|Makefiles & Build Systems]]**
   - Writing a Makefile
   - Rules and Targets

6. **[[06-Libraries|Libraries]]**
   - Static (`.a`) vs Dynamic (`.so`/`.dll`)

7. **[[07-Type-Qualifiers|Type Qualifiers]]**
   - `volatile`
   - `restrict`

8. **[[08-C-Standards|C Standards History]]**
   - K&R, C89, C99, C11, C23

9. **[[09-CMake-Basics|CMake Basics]]**
   - `CMakeLists.txt`
   - Building and Linking

10. **[[11-C23-Features|C23 Features (New)]]**
    - `nullptr`, `constexpr`, `auto`
    - Standard Attributes (`[[nodiscard]]`)
    - `_BitInt` and Binary Literals
