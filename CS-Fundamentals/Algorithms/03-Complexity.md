# Big O Complexity

-   **O(1)**: Constant time (Array access).
-   **O(log n)**: Logarithmic (Binary Search).
-   **O(n)**: Linear (Looping through list).
-   **O(n log n)**: Linearithmic (Merge Sort).
-   **O(n^2)**: Quadratic (Nested loops).

## Space Complexity
C gives you direct control over memory, so optimizing space is often as important as time.
-   **Stack Space**: Recursion depth (O(n) for naive recursion).
-   **Heap Space**: `malloc` usage.
