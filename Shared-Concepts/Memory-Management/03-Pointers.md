# Pointers and References

A pointer is a variable that stores a memory address.

## Concept
-   **Value**: The data itself (e.g., integer `42`).
-   **Address**: The location in memory (e.g., `0x7ffee4`).
-   **Pointer**: A variable holding `0x7ffee4`.

## Usage
-   **Indirection**: Accessing value via address.
-   **Dynamic Allocation**: Managing heap memory.
-   **Data Structures**: Linking nodes in a list/tree.

## Risks
-   **Dangling Pointer**: Pointing to freed memory.
-   **Null Pointer**: Pointing to `0x0` (invalid).
-   **Buffer Overflow**: Writing past the allocated boundary.
