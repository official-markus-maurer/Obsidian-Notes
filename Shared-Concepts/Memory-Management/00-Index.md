# Memory Management

Memory management is the process of controlling and coordinating computer memory.

## Key Concepts

1.  **[[01-Stack-vs-Heap|Stack vs Heap]]**
    -   Stack: Static, fast, automatic cleanup.
    -   Heap: Dynamic, slower, manual cleanup.
2.  **[[02-Virtual-Memory|Virtual Memory]]**
    -   Address translation.
    -   Paging and Segmentation.
3.  **[[03-Pointers|Pointers]]**
    -   Direct memory access.
    -   References in high-level languages.
